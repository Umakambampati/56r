Introduction to Python

1.What is Language?

A:A language is a system of communication using symbols, words, or signs. It helps people express thoughts, ideas, or instructions to others.

2.What is Programming Language?

A: programming language is a formal way to give instructions to a computer. It allows developers to write code that a computer can understand and execute.

3.Why Python is used?

A: 1. Ease of Use: Simple syntax, ideal for beginners.

2. Versatility: Supports multiple programming paradigms (object-oriented, procedural,functional).

3. Extensive Libraries: Rich ecosystem for tasks like data analysis, machine learning, and web development.

4. Community Support: Large and active community for troubleshooting and learning.

5.Cross-Platform: Python programs can run on various platforms without modification(Windows, macOS, Linux).

4.What is Python?

A: Python is a high-level, interpreted programming language known for its simplicity and readability

5.Who created Python, and when was it first released?

A: It was created by Guido van Rossum in the late 1980s and first released in 1991 . Python emphasizes code readability and supports multiple programming paradigms, including object-oriented, procedural programming, functional programming.

6. In which domains is Python generally utilized?

A:1.Web development: Frameworks: Django,flask

2.Data Science and Machine Learning: Predictive modeling and data analysis using NumPy, Pandas, TensorFlow, scikit-learn.

3. Automation(Scripting): Automating repetitive tasks.

4. Game Development: Framework: PyGame.

5. Scientific Computing: Libraries: SciPy, Matplotlib.

6. Artificial Intelligence (AI) and Deep Learning: Libraries: Keras, PyTorch.

7.Web Scraping: Libraries: BeautifulSoup, Scrapy.

8. Cybersecurity: Used for creating tools to test system vulnerabilities.

7. How does Python handle memory, and what is the downside?

A: Python uses dynamic typing and garbage collection, which increases memory usage. This makes it less suitable for memory-critical applications.

8. What is GIL in Python, and why is it a limitation?

A: GIL stands for Global Interpreter Lock, which allows only one thread to execute at a time. This limits multi-threading performance, especially on multi-core processors.

9. Which big companies use Python?

A: Major companies using Python include Google, Netflix, Instagram, Spotify, NASA, and Uber. It's also used in dashboards like 10000 Coders and platforms like Reddit.

10.What are some other disadvantages of python?

A:

1. Slower Execution Speed

○ Interpreted language , making it slower than compiled languages like C++ or Java.

○ Not ideal for performance-critical applications (e.g., real-time systems, gaming engines).

2.Runtime Errors

○ Python''s dynamic nature allows runtime errors to occur if code is not thoroughly tested.

○ This can lead to bugs in production if not handled properly.

3.Limited Support for Low-Level Programming

○ Languages like C or Rust are better suited for such tasks.

11. How is Python interpreted and what does that mean?

A: Python code is executed line by line by the Python interpreter. Being interpreted means you don’t need to compile the code — it runs directly, making debugging easier and faster.
